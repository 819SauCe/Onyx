# Code of Conduct | Código de Conduta

[English](#english) | [Português](#português)

---

## English

### Our Pledge

We want to make Jesterx a welcoming space for everyone.  No matter your experience level, background, or identity - if you're here to contribute and learn, you're welcome.

### Our Standards

**Good behavior:**

- Be respectful and considerate
- Accept constructive feedback
- Focus on what's best for the project
- Help others when you can
- Be patient with newcomers

**Not cool:**

- Harassment or intimidation of any kind
- Trolling or inflammatory comments
- Personal or political attacks
- Publishing others' private information
- Spamming issues or PRs
- Any conduct that would be inappropriate in a professional setting

### Our Responsibilities

Project maintainers are responsible for clarifying standards and will take appropriate action in response to any unacceptable behavior.

We have the right to remove, edit, or reject comments, commits, code, issues, and other contributions that don't align with this Code of Conduct. 

### Scope

This Code of Conduct applies in all project spaces and when someone is representing the project publicly. 

### Enforcement

If you experience or witness unacceptable behavior, please report it by opening an issue or contacting the project maintainers directly.

All complaints will be reviewed and investigated.  We're committed to maintaining confidentiality regarding the reporter. 

### Consequences

Depending on the severity: 

1. **Warning** - A private warning explaining what was inappropriate
2. **Temporary Ban** - No interaction with the project for a specified period
3. **Permanent Ban** - Permanent removal from all project spaces

### Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.0.

## Português

### Nosso Compromisso

Queremos fazer do Jesterx um espaço acolhedor pra todo mundo. Não importa seu nível de experiência, background ou identidade - se você tá aqui pra contribuir e aprender, é bem-vindo.

### Nossos Padrões

**Comportamento esperado:**

- Seja respeitoso e atencioso
- Aceite feedback construtivo
- Foque no que é melhor pro projeto
- Ajude os outros quando puder
- Seja paciente com iniciantes

**Não é legal:**

- Assédio ou intimidação de qualquer tipo
- Trollagem ou comentários inflamatórios
- Ataques pessoais ou políticos
- Publicar informações privadas de outras pessoas
- Spam em issues ou PRs
- Qualquer conduta que seria inapropriada em um ambiente profissional

### Nossas Responsabilidades

Os mantenedores do projeto são responsáveis por esclarecer os padrões e vão tomar ações apropriadas em resposta a qualquer comportamento inaceitável.

Temos o direito de remover, editar ou rejeitar comentários, commits, código, issues e outras contribuições que não estejam alinhadas com este Código de Conduta.

### Abrangência

Este Código de Conduta se aplica em todos os espaços do projeto e quando alguém estiver representando o projeto publicamente. 

### Aplicação

Se você presenciar ou sofrer algum comportamento inaceitável, reporte abrindo uma issue ou contatando os mantenedores do projeto diretamente.

Todas as reclamações serão revisadas e investigadas. Mantemos sigilo sobre quem reportou. 

### Consequências

Dependendo da gravidade:

1. **Aviso** - Um aviso privado explicando o que foi inapropriado
2. **Ban Temporário** - Sem interação com o projeto por um período específico
3. **Ban Permanente** - Remoção permanente de todos os espaços do projeto

### Atribuição

Este Código de Conduta é adaptado do [Contributor Covenant](https://www.contributor-covenant.org), versão 2.0.
